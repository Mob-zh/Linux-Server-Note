## 9.2 poll

```cpp
#include＜poll.h＞
int poll(struct pollfd*fds,nfds_t nfds,int timeout);


struct pollfd
{
    int fd;/*文件描述符*/
    short events;/*注册的事件*/
    short revents;/*实际发生的事件，由内核填充*/
};
```

### LT和ET模式
LT:高效的poll，可以到下一次wait再响应
ET:高效epoll模式，立即处理ET模式的响应事件
