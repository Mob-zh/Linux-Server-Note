# 十、信号

## 10.1 Linux信号概述

### 10.1.1 发送信号

```c
#include <sys/types.h>
#include <signal.h>
int kill(pid_t pid,int sig);

/*
para:

    pid > 0 信号发送给pid进程
    pid = 0 信号发送给本进程组内的其他进程
    pid = -1 信号发送给除了init进程外的所有进程（要有权限）
    pid < -1 信号发送给组id为-pid的进程组的所有成员

ret:
    EINVAL 无效的信号
    RPERM 没有权限
    RSRCH 目标不存在
*/

```
